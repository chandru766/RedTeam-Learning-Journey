## 🧠 What I Learned

### 🧱 OSI Model (7 Layers)

1. **Physical** – Transmission of raw bits  
2. **Data Link** – MAC addressing and frame transmission  
3. **Network** – Routing (IP addressing)  
4. **Transport** – End-to-end communication (TCP/UDP)  
5. **Session** – Manages sessions  
6. **Presentation** – Data formatting and encryption  
7. **Application** – User interaction (HTTP, FTP, etc.)  

---

### 🌐 TCP/IP Model (4 Layers)

1. **Network Interface**  
2. **Internet**  
3. **Transport**  
4. **Application**  

👉 Simpler model used in real-world networking  

---

### 🔗 TCP (Transmission Control Protocol)
- Connection-oriented protocol  
- Reliable and ensures data delivery  
- Used in HTTP, HTTPS  

---

### ⚡ UDP (User Datagram Protocol)
- Connectionless protocol  
- Faster but less reliable  
- Used in streaming, DNS  

---

### 📡 DHCP (Dynamic Host Configuration Protocol)
- Automatically assigns IP addresses to devices  
- Reduces manual configuration  

---

### 🌍 IP Address Classes

- **Class A** – Large networks  
- **Class B** – Medium networks  
- **Class C** – Small networks  

---

### 🔢 Subnetting
- Dividing a network into smaller sub-networks  
- Improves performance and security  

---

### 🧩 Network Part & Host Part
- **Network Part** → Identifies the network  
- **Host Part** → Identifies devices in the network
