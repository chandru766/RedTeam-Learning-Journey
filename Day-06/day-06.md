## 🧠 What I Learned

### 🔥 What is a Firewall

* A firewall is a **security system** that monitors and controls network traffic
* It acts as a **barrier between trusted and untrusted networks**

👉 Example: Protects internal network from the internet

---

### ⚙️ How Firewall Works

* Filters incoming and outgoing traffic
* Uses rules to **allow or block data packets**
* Based on:

  * IP address
  * Port number
  * Protocol

👉 Only trusted traffic is allowed

---

### 🛡️ Types of Firewalls

* **Packet Filtering Firewall**
  → Filters packets based on rules

* **Stateful Firewall**
  → Tracks active connections

* **Proxy Firewall**
  → Acts as an intermediary

* **Next-Generation Firewall (NGFW)**
  → Advanced features like deep inspection

---

### ❗ Why Firewall is Important

* Protects systems from unauthorized access
* Blocks malicious traffic
* Helps prevent cyber attacks
* Controls network activity

---

### 🚨 What is IDS (Intrusion Detection System)

* IDS monitors network/system for **suspicious activity**
* Alerts when a potential attack is detected

👉 It does NOT block — only detects

---

### ⚙️ How IDS Works

* Analyzes traffic and system behavior
* Compares with known attack patterns
* Generates alerts for suspicious activity

---

### 🧩 Types of IDS

* **Network-based IDS (NIDS)**
  → Monitors entire network traffic

* **Host-based IDS (HIDS)**
  → Monitors a single system

---

## 🛠️ Practical Understanding

* Learned difference between firewall and IDS
* Understood how traffic filtering works
* Got idea of detecting vs preventing attacks
* Learned basic network security concepts

---

## 💡 Key Takeaways

* Firewall = **Prevention (blocks attacks)**
* IDS = **Detection (alerts attacks)**
* Both are important for network security
* Different types of firewalls provide different protection levels
