## 🧠 What I Learned

### 🔐 Information Security vs Cybersecurity
- **Information Security** focuses on protecting all types of data (digital and physical)  
- **Cybersecurity** focuses on protecting systems, networks, and digital data  

---

### 🌐 Application Security
- Protecting applications from vulnerabilities and attacks  
- Ensures secure coding and safe user data handling  

---

### 🖧 Network Security
- Protecting network infrastructure from unauthorized access  
- Includes monitoring traffic and preventing attacks  

---

### 🔥 Firewalls
- A firewall acts as a barrier between trusted and untrusted networks  
- It filters incoming and outgoing traffic based on rules  

---

### 🔑 Authentication
- Process of verifying the identity of a user  
- Example: Username and password  

---

### 🛡️ Authorization
- Determines what actions a user is allowed to perform  
- Example: Access control (admin vs user)  

---

### 🧾 Accountability
- Tracking user actions in a system  
- Helps in auditing and identifying malicious activity  

---

### 🆔 Identification
- Claiming an identity (e.g., entering a username)  

---

### 🔐 Types of Authentication Factors

1. **Singal factor authentication**  
   - Password or PIN  

2. **Two factor authentication**  
   - OTP, security token  

3. **Multi-factor authentication**  
   - Fingerprint, face recognition, OTP, security token  

---

## 🛠️ Practical Understanding

- Learned how authentication and authorization work together

- Gained basic knowledge of security principles  

---

## 💡 Key Takeaways

- Security is divided into multiple domains (application, network, etc.)  
- Authentication and authorization are core concepts in cybersecurity  
- Firewalls are essential for protecting systems
